# Oneri

Bu repo, Swift ile yazılmış iOS uygulaması **Oneri**’yi içerir.  
Bu proje, kullanıcı kayıt işlemleri, öneri gönderme ve yerel veri saklama işlevlerini içerir.



- Kullanıcı Modelleri (User)
- Soru Modelleri (Soru)
- Persistence Helpers (UserDefaults yardımıyla veri saklama)
- UIKit tabanlı ViewController’lar

- Kullanıcı bilgilerini saklama
- Soru & öneri ekleme ve listeleme
- Basit UI yapısı
- UserDefaults ile yerel veri depolama

- Swift
- UIKit
- MVC Mimari Deseni
- UserDefaults
